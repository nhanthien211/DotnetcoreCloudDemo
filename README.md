## How to run with VS



#### Step 1: 

``
Install PostgreSQL 11 from official site
``

#### Step 2: 

``
Restore DB using seed.sql
``

#### Step 3: 

``
Open solution and edit connection string username and password in appsetting.json (DB name if needed)
``
